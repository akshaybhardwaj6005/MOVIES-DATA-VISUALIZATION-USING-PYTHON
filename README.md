📌 Overview
This project performs exploratory data analysis (EDA) on a movie dataset to uncover insights related to budget, revenue, genres, and ratings. The analysis is done using Python libraries like Pandas, Matplotlib, and Seaborn.
📂 Dataset
Dataset used: tmdb_5000_movies.csv
Contains information about:
Movie titles
Budget
Revenue
Genres
Ratings
And more
🛠️ Technologies Used
Python 🐍
Pandas
NumPy
Matplotlib
Seaborn
Jupyter Notebook
🔍 Key Analysis Performed
1. Data Loading & Cleaning
Loaded dataset using Pandas
Checked shape, columns, and missing values
Cleaned and prepared data for analysis
2. Budget Distribution
Visualized how movie budgets are distributed using plots
3. Budget vs Revenue
Scatter plot to analyze relationship between:
Movie budget 💰
Movie revenue 📈
4. Average Rating per Genre
Extracted genres
Calculated average ratings for each genre
5. Top 10 Movies by Revenue
Identified highest earning movies
6. Genre Distribution
Pie chart showing genre proportions
Donut chart for better visualization
📊 Visualizations
The project includes:
Histogram (Budget distribution)
Scatter Plot (Budget vs Revenue)
Bar Charts
Pie Chart & Donut Chart (Genre distribution)
🚀 How to Run the Project
Clone the repository:
Bash
git clone https://github.com/your-username/movie-data-analysis.git
Navigate to the project folder:
Bash
cd movie-data-analysis
Install dependencies:
Bash
pip install pandas numpy matplotlib seaborn
Run the notebook:
Bash
jupyter notebook movie_project.ipynb
